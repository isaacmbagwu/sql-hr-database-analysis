## HR Database Analysis

## 📌 Overview

This project analyzes an HR database using SQL to explore employee distribution, salary structures, and workforce insights. It demonstrates practical data analysis skills using real-world business questions.

🛠 Tools Used
SQL (MySQL)

The dataset includes:

Employees
Departments
Payroll
Employee locations

## 🔍 Key Analysis Performed
Employee and department relationships using JOINs
Salary and bonus analysis
Workforce distribution across departments
Age demographics and workforce structure
Regional staff concentration
Compensation pattern analysis

## 📊 Sample Query
-- Top 5 highest-paid employees
SELECT 
    emp_name,
    emp_role,
    emp_salary
FROM employees
ORDER BY emp_salary DESC
LIMIT 5;

## 💡 Key Insights
1. HR department appears to have the highest staff concentration
2. Salary differences suggest role-based compensation structure
3. High bonus percentages may indicate performance-based roles
4. Some locations have significantly more employees than others
